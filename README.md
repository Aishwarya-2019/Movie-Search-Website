Movie Search App
A simple web application that lets users search for movies using the OMDB API. Users can view basic information about each movie in a grid of cards, click a card to see more details in a modal, and navigate through multiple pages of results. Additional features include search history, dark/light mode toggle, and responsive design.

Features
Search movies by title (with Enter key support)
Display search results as clickable cards
View detailed movie information in a modal dialog
Pagination for browsing large result sets
Recent search history stored in localStorage (last 5 searches)
Dark and light theme with toggle and persisted preference
Smooth hover and animation effects
Responsive layout with improved typography and spacing
Setup
Clone the repository or download the source.
Open index.html in a modern web browser.
Enter a movie title in the search box and click Search or press Enter.
No build tools or dependencies are required; this is a vanilla HTML/CSS/JavaScript project.

Usage
Search: Type a movie name and press Enter or click the search button.
Pagination: Use the previous/next buttons at the bottom of the results to navigate pages.
History: Click on any recent search term to repeat the search.
Theme: Click the toggle button at the top-right to switch between dark and light modes.
Details: Click a movie card to open a modal with more information (genre, rating, director, actors, plot).
